# NAME

Alien::LibGumbo - Gumbo parser library

# DESCRIPTION

This distribution installs [libgumbo](https://codeberg.org/gumbo-parser/gumbo-parser) on your
system for use by perl modules like [HTML::Gumbo](https://metacpan.org/pod/HTML::Gumbo).

The original [libgumbo](https://github.com/google/gumbo-parser) by Google was
archived on GitHub with no development since 2016. This distribution now uses
the maintained fork by Grigory Kirillov, whose first release of the fork
(0.11.0) was in July 2023.

**If you're interested in parsing HTML** then you want [HTML::Gumbo](https://metacpan.org/pod/HTML::Gumbo) module, not this.
